# Tobik Web Gallery 5000
Blazor rewrite of the php software Tobik Web Gallery 3000
Supports:
* Normal photos
* Videos
* Live photos from iPh*ne or Android
* Sound photo from old samsung phones
* Stereoscopic 3D Photos from Nintendo 3DS*
* 3D Models in .glb format
* 360 Photosphere from Google Camera/GCam
* Password management of photo gallery

Demo: https://twg.tobikcze.eu/gallery/demo/   
Password encryption demo: https://twg.tobikcze.eu/gallery/demopsw/     
the password is Demo123!

To enable Live photo support, create "enablelive" in the directory
To enable sound photo support, create "enablewav" in the directory
To enable 3D photo support, create "enable3ds" in the directory

# Open source usage
this software relies on the following code:
* Three.js
* Photo sphere viewer js
* Metadata extractor
* \<model-viewer>
